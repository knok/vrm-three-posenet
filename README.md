# VRM Three.js PoseNet Sample

VRM ファイルを PoseNet で検出した姿勢を反映するサンプルです。

## デモ

ウェブカメラの付いたPCで[こちら](https://knok.github.io/vrm-three-posenet/)にアクセスしてください。

※読み込みに少々時間がかかりますのでご注意ください
